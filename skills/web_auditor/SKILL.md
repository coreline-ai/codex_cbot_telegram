# Web Auditor Skill (`web_auditor`)

This is the final gatekeeper for quality. It performs headless browser testing to ensure the site is production-ready.

## Capabilities
- **Viewport Testing**: Checks layout at 375px (Mobile), 768px (Tablet), and 1440px (Desktop).
- **SEO Check**: Validates Meta Title, Description, and Heading hierarchy.
- **Link Validator**: Ensures all internal and external links are functional.
- **Performance Report**: Estimates page weight and suggests optimizations.
