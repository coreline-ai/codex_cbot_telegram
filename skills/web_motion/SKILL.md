# Web Motion Design Skill (`web_motion`)

This skill adds a layer of "life" to static landing pages using CSS animations and JavaScript libraries like GSAP.

## Capabilities
- **Scroll Reveals**: Elements fade or slide in as the user scrolls.
- **Micro-interactions**: Subtle hover effects on buttons and cards.
- **Glassmorphism Shimmer**: Adds a dynamic light-sweep effect to glass cards.
- **Parallax Backgrounds**: Simple, performant depth effects.
