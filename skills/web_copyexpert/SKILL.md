# Web Copywriting Expert Skill (`web_copyexpert`)

This skill transforms generic text into persuasive marketing copy. It analyzes industry trends (via `web_recon` data) to generate headlines, cross-sells, and CTAs that convert.

## Capabilities
- **AIDA Framework**: Generates content following Attention, Interest, Desire, and Action.
- **Tone Matching**: Adapts style to "Professional", "Friendly", "Hype", or "Luxurious".
- **Keywords**: Integrates high-performing SEO keywords found during reconnaissance.
